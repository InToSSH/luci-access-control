Luci-access-control

OpenWrt internet access scheduler

View on GitHub Download .zip Download .tar.gz

This Luci module for OpenWrt routers allows to block internet access for specific clients. You may also define schedules to control internet access at desired time. Everything is configurable via simple GUI interface.
